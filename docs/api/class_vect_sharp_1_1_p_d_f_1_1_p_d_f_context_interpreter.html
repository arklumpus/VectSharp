Convert the document to a PDFDocument representation.

Parameters
documentThe Document to convert.
textOptionDefines whether the used fonts should be included in the PDF document.
compressStreamsIndicates whether the streams in the PDF document should be compressed.
linkDestinationsA dictionary associating element tags to link targets. If this is provided, objects that have been drawn with a tag contained in the dictionary will become hyperlink to the destination specified in the dictionary. If the destination starts with a hash (#), it is interpreted as the tag of another object in the current document; otherwise, it is interpreted as an external URI.
outlineDocument outline (table of contents).
metadataDocument metadata. Use PDFMetadata.NoMetadata() if you do not wish to include metadata in the document.
