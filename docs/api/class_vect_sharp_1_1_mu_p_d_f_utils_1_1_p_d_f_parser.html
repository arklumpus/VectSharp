Import the specified page from a document.

Parameters
fileNameThe path to the document file.
pageNumberThe page number (starting at 0).
linkDestinationsWhen this method returns, this variable will contain a dictionary used to associate graphic action tags to hyperlinks. This can be used to enable such links when rendering the Page to a file.
textOptionThis parameter determines whether text elements are rendered as paths (which provides more accurate results) or as text.
includeAnnotationsWhether annotations (e.g., signatures) should be included in the imported document.
Returns
A Page containing a representation of the specified page from the document.
