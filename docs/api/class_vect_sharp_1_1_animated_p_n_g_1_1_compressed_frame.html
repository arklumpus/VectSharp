Creates a new CompressedFrame from raw pixel data in RGB or RGBA format, applying inter-frame compression based on another frame.

Parameters
rawFrameDataA pointer to the raw pixel data for the new frame.
otherFrameDataA pointer to the raw pixel data for the frame to use as a reference. This can either be the first frame in the animation, or the frame immediately preceeding the current frame.
isFirstFrameIf otherFrameData is the raw pixel data for the first frame in the animation, set this to true; if it is the immediately preceeding frame, set this to false.
widthThe width of the image in pixels.
heightThe height of the image in pixels.
hasAlphaIf the image data is in RGBA format, set this to true; if it is in RGB format, set it to false.
durationThe duration of the frame in milliseconds.
