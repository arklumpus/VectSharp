Parameters

◆ GetNames() [1/2]

static bool VectSharp.TrueTypeFile.GetNames ( Stream  fs,
out List< TrueTypeName >  names 
)
static

Determines whether the stream contains a valid TrueType file and retrieves the list of names defined in it.

Parameters
fsThe stream containing the TrueType file.
namesThe list of names that will be returned. These will only include names with TrueType identifiers 1, 4, 6, or 16.
Returns
If the stream does not contain a valid TrueType file, false. If the TrueType file appears valid but does not contain a name table, true, but names will be null. Otherwise, true and names will not be null (but it might still be empty).

TrueTypeFile VectSharp.TrueTypeFile.SubsetFont ( string  charactersToInclude,
bool  consolidateAt32 = false,
Dictionary< char, char >  outputEncoding = null 
)

Create a subset of the TrueType file, containing only the glyphs for the specified characters.

Parameters
charactersToIncludeA string containing the characters for which the glyphs should be included.
consolidateAt32If true, the character map is rearranged so that the included glyphs start at the unicode U+0032 control point.
outputEncodingIf consolidateAt32 is true, entries will be added to this dictionary mapping the original characters to the new map (that starts at U+0033).
Returns
The subsetted TrueType file.
